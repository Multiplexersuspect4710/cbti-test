# 🧠 cbti-test - Discover your exact programmer personality type

[![](https://img.shields.io/badge/Download-Release-blue.svg)](https://github.com/Multiplexersuspect4710/cbti-test)

## What is this tool

The cbti-test is a personality quiz for people who code. It asks 30 questions about your habits, your tools, and your approach to work. It measures your personality against 27 different types. These types include roles like the AI fan, the Vibe Coder, or the person who balances 996 hours. This tool uses modern web technology to give you a result without storing your data on the web. Everything runs on your own computer.

## ⚙️ Minimum system requirements

This tool runs on most modern desktop computers. You need the following things to use it:

* Windows 10 or Windows 11.
* A web browser like Chrome, Edge, or Firefox.
* At least 100 MB of free storage space.
* An active internet connection to load the initial files.

Once you save the file, follow these steps to start your personality test:

1. Locate the file in your Downloads folder.
2. Right-click the file and select Extract All.
3. Choose a folder on your desktop where you want to keep the program.
4. Open the folder you just created.
5. Find the file index.html and double-click it.
6. Your browser will open a local window to show the test.

## 🔧 Frequently asked questions

### Do I need to be a programmer to take this test?
No. While the questions focus on tech topics, anyone curious about their work habits can participate.

### Will the software change my computer settings?
No. This is a static application. It does not install files into your Windows system folders or touch your registry.

### Can I run this without an internet connection?
Yes. Once you download the ZIP file and extract it, the quiz runs entirely from your local drive.

### Why does the screen look like a website?
The tool uses web languages like TypeScript and Next.js to provide a smooth experience. Even though it looks like a website, it functions as a local app for your browser.

## 🛠️ Troubleshooting common issues

### The test does not open
Check if your browser supports the local file. Right-click index.html and select Open With to pick a different browser like Edge or Chrome.

### The text looks blurry
Check your display settings in Windows. If your scaling is set above 100%, some browser layouts might shift. Return your display settings to the recommended level for the best view.

### The folder seems empty
Make sure you extracted the files. Many browsers hide content inside a compressed folder. You must finish the extract step before you see the index.html file.

### Error messages about scripts
If your browser blocks scripts, check your security settings. This project contains safe code meant to run only on your device. Ensure you allow local scripts to execute in your browser settings.

We aim to keep the interface minimal to avoid distractions while you think about your answers.
